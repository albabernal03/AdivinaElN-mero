Hemos resuelto un juego de adivinar un número, que se encuentra en un determinado rango dependiendo del nivel seleccionado. En total hay cuatro niveles:
1. Nivel (0-100) 
2. nivel (0-1000)
3. Nivel (0-1000000)
4. Nivel (0-1000000000000)
